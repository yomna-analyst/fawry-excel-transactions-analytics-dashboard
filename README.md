# 📊 Fawry Transactions Analytics Dashboard (Excel)

Analytics dashboard built using **Microsoft Excel (Pivot Tables & Charts)** to analyze transaction performance, failure rates, and service efficiency.

---

## 🎯 Objective
Monitor transaction success and failure patterns to improve service performance and reduce revenue loss.

---

## 📊 Dashboard Overview

- Top & Least Performing Services  
- Failure Reasons Analysis  
- Failure Revenue Ratio by Company  
- Success vs Failure Trends Over Time  
- Wallet vs Bank Transactions  
- API vs Web Performance  

---

## 🎛 Filters
- Company  
- Industry  
- Month  

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  

---

## 💡 Key Insights
- Technical errors are the main cause of failures  
- Some companies have higher failure impact on revenue  
- Performance differs between API and Web channels  
- Service usage varies significantly  

---

## 🚀 Outcome
Provides clear visibility into transaction performance and helps identify areas for improvement.
